# Cross-Ailumette

![demo](https://i.ibb.co/P5FC4v0/cross-ailumette.png)

###### 1. To install the node_modules :
```shell
$ npm install
```

###### 2. To start the game in shell :
```shell
$ node run.js
```
Cross :  
```shell
$ node run.js --gui
```
###### 3. To start the with Electron :
```shell
$ npm run start
```

Enjoy the game !
